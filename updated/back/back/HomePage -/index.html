It involves designing and building the visible part of a website that users interact with directly. It requires a combination of design and technical skills to create visually appealing and functional web experiences.

4) JavaScript is a programming language used for adding interactivity and dynamic behavior to web pages. It enables features like form validation, animations, user interactions, and data manipulation. JavaScript can be embedded directly in HTML or included through external scripts
